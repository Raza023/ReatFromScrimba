1. What is a React component?
Ans) composable code that can be reused in ou react application.




2. What's wrong with this code?
```
function myComponent() {
    return (
        <small>I'm tiny text!</small>
    )
}
```
Ans) Can't use ' in jsx as it gives error: "Error: Cannot parse given Error object".
Should not use camel case in custom component name. Should use Pascal case like <MyComponent />




3. What's wrong with this code?

function Header() {
